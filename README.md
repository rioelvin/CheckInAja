# CheckInAja
- Deskripsi Aplikasi :  
Aplikasi ini adalah aplikasi untuk melihat rating atau preview hotel

- Fungsi :  
Pengguna dapat mengetahui foto, fasilitas, dan deskripsi akomodasi, serta rating dan komentar

- Fitur :  
  - Splash Screen  
  - Search Bar
  - Pilihan Hotel

  - Map
